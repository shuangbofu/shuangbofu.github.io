Docker基础指令

May 24 2018

Docker 是一个开源的应用容器引擎，让开发者可以打包他们的应用以及依赖包到一个可移植的容器中，然后发布到任何流行的 Linux 机器上，也可以实现虚拟化。容器是完全使用沙箱机制，相互之间不会有任何接口。

以下是Docker的基础指令

拉取镜像

docker pull 镜像

例如：

docker pull centos:7
docker pull mysql:5.7

列出镜像

docker images

结果如下：

删除镜像

docker rmi 镜像

例如：

docker rmi 2d194b392dd1
docker rmi centos:7

导入导出镜像

导出镜像

docker save 镜像 >/root/xx.tar.gz

例如：

docker save 5d4d51c57ea8 > /root/mysql5.7.tar.gz

导入镜像

docker load 镜像 </root/xx.tar.gz

例如：

docker save 5d4d51c57ea8 < /root/mysql5.7.tar.gz

启动容器

以交互方式启动容器（前台启动）

docker run -it –name 容器名称 镜像

例如：

docker run -it –name mycentos 2d194b392dd1 /bin/bash

docker run -it –name mytomcat tomcat:8 -p8080:8080 /bin/bash

以守护进程方式启动容器（后台启动）

docker run -d –name 容器名称 镜像

例如：

docker run –name test-mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.7

查看容器

查看正在运行容器

docker ps

结果如下：

查看历史运行过的容器

docker ps -a

查看最近运行过的容器

docker ps -l

停止/重启容器

docker stop/restart 容器名称或者容器ID

例如：

docker stop mycentos
docker restart test-mysql

删除容器

删除指定容器

docker rm 容器名称或者容器ID

删除所有容器

docker rm ‘docker ps -a -q’
